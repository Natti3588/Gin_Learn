# リダイレクト

Gin は HTTPリダイレクト（クライアントを異なるURLに送信する）とルーターリダイレクト（往復通信なしにリクエストを内部的に別のハンドラに渡す）のサポートをしている。

## HTTPリダイレクト

`c.Redirect` の第一引数に適切なHTTPステータスコードをセットして、第二引数にリダイレクト先をセットする

- 301（`http.StatusMovedPermanently`） リソースが恒久的に移動。 ブラウザと検索エンジンはキャッシュを更新する。
- 302（`http.StatusFound`） 一時的なリダイレクト。 新しいURLはキャッシュしない
- 307（`http.StatusTemporaryRedirect`） 302と同じように一時的にリダイレクト。　POSTメソッドなどのリクエスト情報を保持したまま転送できる
- 308（`http.StatusPermanentRedirect`） 301と同様に恒久的にリダイレクト。 POSTメソッドなどを 307と同じように保持ができる
