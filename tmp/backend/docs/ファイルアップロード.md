# ファイルアップロード

`Gin`でのファイルアップロードは、Go標準ライブラリの`multipart/form-data`処理を
`gin.Context`のメソッドで扱いやすくラップしたものです。
主に以下の三つのメソッドが使われます。

- `c.FormFile(name)` フォームフィールド名で単一のファイルをリクエストから取得します
- `c.MultipartForm()` multipartフォーム全体をパースし、アップロードされたすべてのファイルとフィールド値にアクセスできます。
- `c.SaveUploadedFile(file, dst)` 受信したファイルをディスク上の指定パスに保存する便利なメソッド

## セキュリティ上の注意

クライアントが報告するファイル名（`file.Filename`）は信頼すべきではない。
ファイルシステム操作で使用する前に、必ずサニタイズまたは置換をする。

---

**サニタイズ**とは「システムにとって危険な文字やコードを検知して、無害な文字列に変換する処理」のこと
